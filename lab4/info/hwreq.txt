Написать программу на ассемблере masm32 (для архитектуры, заданной командой .386) по заданию: 
"
Написать программу, которая находит результат функции y = y1 mod y2 для каждого x. 
Сложные функции y1 и y2 указаны далее:
y1 = {
    8 + abs(x), если x < 1; 
    abs(a) * 2, если x >= 1
},
y2 = {
    3, если x = a;
    a + 1, если x != a
}.
Переменная a = 3 задается в программе и не меняется в процессе ее исполнения.
Переменная x изменяется в диапазоне от 1 до 15 с шагом 1.
При выполнении работы можно использовать команды условных переходов и организации циклов.
Для вывода в консоль использовать crt_printf из библиотеки msvcrt.lib.
Подключения производить по шаблону: include includes\... .
Результат должен выводиться в в 16-тиричном и 10-тичном виде.
Решение должно быть организовано с помощью цикла FOR, реализованого с помощью команды loop.
Код писать в процедуре 
".code
solve PROC
...
solve ENDP
end solve
"
Программу объясните"
